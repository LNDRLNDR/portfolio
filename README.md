# Custom Dynamic Broadcasting Overlay for Rocket League

Portfolio of fully custom, fully automated broadcasting overlays and game HUDs built for Rocket League teams and tournaments. Each overlay matches a client's brand and can include in-game HUDs, goal screens, post-game stats, series stats, caster screens, MVP screens, and intermission screens.

**[View the live page](https://lndrlndr.github.io/portfolio/)**

## Featured projects

- RSC EU: full broadcast package: game HUD, goal overlay, win screen, post-game stats, series stats, MVP, caster, intermission
- SaltMine3 (Johnnyboi): 1v1 overlay with demo respawn timer, in-game stat pop-ups, and replay minimap
- Gen.G Mobil1 Racing: Formula 1 theme with speed meter, animated sponsor boxes, and goal-replay minimap
- PSG Tundra, Quadrant, NRG Combine, MLE, UniFlick, Flip & Spin, The Draw, Porsche 911 Invitational, Rocket Street LATAM, Svenska Raketligan, and more
